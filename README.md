# リンク一括ダウンローダー（Chrome拡張）

選択範囲、または指定した要素の中にあるリンクから、PDFなどのファイルをまとめてダウンロードする拡張機能。

## インストール（未署名・開発者モード読み込み）
1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ `link-file-downloader` を選択

## 使い方
ツールバーの拡張アイコンをクリック → 2つのモード:

- **📝 選択範囲から取得**：先にページ上でテキスト（リンクを含む範囲）をドラッグ選択してから、このボタンを押す。選択範囲内のリンクを収集。
- **🎯 要素を選んで取得**：押すとピッカーが起動。マウスを乗せると要素がハイライトされ、クリックでその要素（ブロック）内の全リンクを収集。Escでキャンセル。

収集結果は画面右上のパネルに一覧表示される。
- 「ファイルのみ」チェックで既知の拡張子（pdf, zip, docx, jpg…）だけ表示
- 拡張子フィルタ欄に `pdf,zip` のように入力で絞り込み
- チェックを付けて「ダウンロード」ボタン

## 仕組み（動かない拡張との違い）
- **Manifest V3** 準拠
- ダウンロードは `chrome.downloads` API（service worker）で実行 → クロスオリジンのPDF等も確実に保存
- 相対URLは絶対URLへ変換、重複除去
- ファイル名重複は `uniquify` で自動連番
- 逐次ダウンロード＋ウェイトでサーバー側のブロックを軽減

## 既知の制限
- ログイン必須・Referer必須のファイルは 403 になることがある（chrome.downloads は Referer を送らないため）
- JavaScriptで動的に開くだけの「href を持たないボタン」は対象外（aタグのhrefのみ収集）
- `chrome://` やChrome Web Store等の特別なページでは動作しない

## ファイル構成
- `manifest.json` — 拡張定義
- `background.js` — ダウンロード実行（service worker）
- `content.js` — リンク収集・要素ピッカー・結果パネルUI
- `popup.html` / `popup.js` — ツールバーのポップアップ
- `icons/` — アイコン
