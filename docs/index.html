リンク一括ダウンローダー

Link File Downloader

ページのリンクから、PDFなどのファイルをまとめてダウンロード。

使い方 / How to use GitHub プライバシー

主な機能

📝 選択範囲から

リンクを含む範囲をドラッグ選択し、その中のリンクだけを収集。

🎯 要素を選んで

ページのブロックをクリックすると、その中の全リンクを収集。

📦 ZIPでまとめて

選んだファイルを1つのZIPに。保存ダイアログは1回だけ。

📁 個別に保存

専用フォルダへ直接保存。ファイル名の重複も自動で回避。

Features (EN)

From a selection

Drag-select a range and collect only the links inside it.

Pick an element

Click a block on the page to collect all links within it.

Download as ZIP

Bundle selected files into a single ZIP — only one save dialog.

Save individually

Save straight into a dedicated folder, with auto de-duplication.
