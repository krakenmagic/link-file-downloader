プライバシーポリシー

拡張機能「リンク一括ダウンローダー（Link File Downloader）」/ 最終更新: 2026-06-26

収集する情報

本拡張機能は、利用者の個人情報を一切収集しません。氏名・メールアドレス・閲覧履歴・位置情報などを取得・保存・送信することはありません。

データの送信

本拡張機能は、外部のサーバーや開発者に対していかなるデータも送信しません。すべての処理は利用者のブラウザ内（ローカル）で完結します。

権限の利用目的

  • downloads … 選択したファイルをブラウザのダウンロード機能で保存するため。
  • scripting / activeTab … 現在開いているタブ上でリンクを収集し、結果パネルを表示するため。
  • ホストへのアクセス … 「ZIPでまとめて」保存する際に、対象ファイルを取得してZIP化するため。取得したファイルは利用者のダウンロードとして保存される以外に利用しません。

第三者への提供

収集する情報が無いため、第三者へ提供する情報はありません。広告・解析（アナリティクス）の類も使用していません。

お問い合わせ

本ポリシーに関するお問い合わせは、GitHubリポジトリのIssue、または開発者までご連絡ください。

Privacy Policy (English)

Extension: “Link File Downloader” / Last updated: 2026-06-26

Information We Collect

This extension does not collect any personal information. It does not gather, store, or transmit your name, email, browsing history, or location.

Data Transmission

This extension does not send any data to external servers or to the developer. All processing happens locally within your browser.

Why We Request Permissions

  • downloads — to save the files you select via the browser's download feature.
  • scripting / activeTab — to collect links on the current tab and show the result panel.
  • Host access — to fetch the selected files and bundle them into a ZIP when you use “Download as ZIP”. Fetched files are used solely to be saved as your download.

Third Parties

No data is shared with third parties. No advertising or analytics are used.

Contact

For questions about this policy, please open an issue on the GitHub repository or contact the developer.
