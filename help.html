リンク一括ダウンローダー

基本の使い方

ツールバーの拡張アイコンをクリックすると、2つのモードが出ます。
  1. 📝 選択範囲から取得
    先にページ上でリンクを含む範囲をドラッグ選択してから押します。選択範囲内のリンクだけを集めます。
  2. 🎯 要素を選んで取得
    押すとピッカーが起動。マウスを乗せると要素が青くハイライトされ、クリックでその中の全リンクを集めます。Esc でキャンセル。
集めた結果は右上のパネルに表示されます。

パネルの操作

  • ファイルのみ … pdf / zip / docx / jpg などの既知ファイルだけ表示
  • 拡張子で絞込 … pdf,zip のように入力で絞り込み
  • 全選択 / 解除、各行のチェックで対象を調整。上部の青いバーをドラッグで移動

2つの保存方法

AZIPでまとめて … 選んだファイルを1つのZIPにまとめて保存。保存ダイアログは1回だけ。

B個別に保存 … ダイアログを出さず ダウンロード/link-downloader/ へ直接保存。

⚠️「個別に保存」で毎回ダイアログが出てしまうとき

Chrome本体の「保存場所を毎回確認する」がオンです。拡張側からは変えられないため、一度だけChrome設定でオフにしてください。

  1. アドレスバーに chrome://settings/downloads を入力して開く
  2. 「ダウンロード前に各ファイルの保存場所を確認する」 を オフ にする

以降は確認なしで ダウンロード/link-downloader/ に入ります。

うまくいかないとき

  • リンクが0件 … リンクが <a href> でなくJSボタンの可能性。別の範囲・要素を試す。
  • ZIPが「全件取得失敗」 … ログイン必須・Referer必須のファイルは取得不可のことがある。「個別に保存」の方が成功しやすい。
  • chrome:// やChrome Web Storeなど特別なページでは動作しません。

Getting started

Click the extension icon in the toolbar to see two modes.
  1. 📝 From selection
    First drag-select a range that contains links on the page, then click this. Only links inside the selection are collected.
  2. 🎯 Pick an element
    Starts a picker. Hover to highlight an element in blue, then click to collect all links inside it. Press Esc to cancel.
Results appear in the top-right panel.

Panel controls

  • Files only — show known files only (pdf / zip / docx / jpg …)
  • Filter by ext — type like pdf,zip to narrow down
  • All / None and per-row checkboxes to choose items. Drag the blue top bar to move the panel.

Two ways to save

ADownload as ZIP — bundle the selected files into a single ZIP. Only one save dialog.

BSave individually — no dialog; saved straight into Downloads/link-downloader/.

⚠️ If "Save individually" still asks every time

Chrome's "Ask where to save each file" setting is on. The extension can't change it, so turn it off once in Chrome settings.

  1. Open chrome://settings/downloads in the address bar
  2. Turn off "Ask where to save each file before downloading"

After that, files go into Downloads/link-downloader/ with no prompt.

Troubleshooting

  • 0 links — the links may be JS buttons, not <a href>. Try a different range or element.
  • ZIP "all unreachable" — login- or Referer-protected files can't be fetched. "Save individually" is more likely to succeed.
  • Special pages like chrome:// or the Chrome Web Store are not supported.
