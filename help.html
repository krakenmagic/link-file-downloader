リンク一括ダウンローダー

選択範囲・指定要素内のリンクから、PDFなどのファイルをまとめて保存

基本の使い方

ツールバーの拡張アイコンをクリックすると、2つのモードが出ます。
  1. 📝 選択範囲から取得
    先にページ上でリンクを含む範囲をドラッグ選択してから、このボタンを押します。選択範囲内のリンクだけを集めます。
  2. 🎯 要素を選んで取得
    押すとピッカーが起動。マウスを乗せると要素が青くハイライトされ、クリックでその要素（ブロック）内の全リンクを集めます。Esc でキャンセル。
集めた結果は画面右上のパネルに一覧表示されます。

パネルの操作

  • ファイルのみ … pdf / zip / docx / jpg などの既知ファイルだけ表示（オフで全リンク表示）
  • 拡張子で絞込 … pdf,zip のように入力するとその種類だけに絞れます
  • 全選択 / 解除、各行のチェックで保存対象を調整
  • パネル上部の青いバーをドラッグで移動できます

2つの保存方法

AZIPでまとめて
選んだファイルを1つのZIPにまとめて保存します。保存ダイアログは1回だけ。フォルダを1度選べば全部そこに入ります。

B個別に保存
ダイアログを出さず、ダウンロード/link-downloader/ フォルダへ直接流し込みます。

⚠️「個別に保存」で毎回ダイアログが出てしまうとき

Chrome本体の設定で「保存場所を毎回確認する」がオンになっています。これは拡張側から変更できないため、一度だけChrome設定でオフにしてください。

  1. アドレスバーに chrome://settings/downloads と入力して開く
    （または ︙メニュー → 設定 → ダウンロード）
  2. 「ダウンロード前に各ファイルの保存場所を確認する」 のスイッチを オフ にする
  3. 必要なら「保存先」を好きなフォルダに設定

以降は「個別に保存」を押すと、確認なしで ダウンロード/link-downloader/ に自動で入ります。

※ ZIP保存だけは仕様上ダイアログが1回出ます（保存先を選べるようにするため）。毎回同じ場所で良ければ上記設定オフで省略できます。

うまくいかないとき

  • リンクが0件 … そのページのリンクが <a href> ではなくJSボタンの可能性。別の範囲・要素を試してください。
  • ZIPが「全件取得失敗」 … ログイン必須・Referer必須のファイルは拡張のfetchがブロックされることがあります。その場合は「個別に保存」の方が成功しやすいです（ブラウザのCookieが効くため）。
  • 大量・大容量 … ZIP化が重いときは選択を分割するか、個別保存を使ってください。
  • chrome:// やChrome Web Storeなど特別なページでは動作しません。
